Templates

These templates (which should have self-explanatory names) are used for sending notification mails to the GIB team or the project manager. In order to have a translation of a template available, you should have a template with the language ISO code as suffix, e.g. newDataSheetNotificationZh would be the Chinese version of the newDataSheetNotification template.
