GIB Grading Tool: Admin Settings

Forms

Almost every form that is part of this tool is based on a FormBuilder form configuration. The FormBuilder allows you to adjust existing forms (e.g. by adding new fields, changing labels). Please keep in mind that you should not change the identifiers of form fields (e.g. the field "First name" of the Project Manager in the dataSheetForm has "projectManagerFirstName" as identifier) because some of the identifiers are hard-coded in the GIB Grading Tool to use the data for adding accounts etc.

In order to make the GIB Grading Tool use a translated form, the form identifier needs to be appended by the ISO code of the according language. E.g. if both "dataSheetForm" and "dataSheetFormEs" exist, "dataSheetFormEs" is used if someone uses to tool in Spanish language. The default language of the tool is english, forms for the default language must not have an ISO code suffix.
